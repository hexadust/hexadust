# 💫 About Me:
I am a Robotics Systems and Mechatronics engineer from Bauman Moscow State Technical University specializing in intelligent automation, control systems, and AI driven robotics. My work integrates mechanical design, embedded systems, smart systems, and advanced control theory to develop an automation and robotics project. My personal projects, such as IoT mobile robots, self balancing robot, robotic manipulators with computer vision, collaborative robot, and IoT monitor device for industrial platform. With strong experience in CAD, AI, IoT, as well as system modeling in MATLAB, I focus on building robust hardware systems enhanced by adaptive software and data driven intelligence. I work across C++, Python, JavaScript, ROS, OpenCV, and modern web technologies to create scalable, real world solutions for industrial and maunfacture applications.
